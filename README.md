# dev-skills

General-purpose development skills for Claude Code: retrospectives, git workflow, investigation methodology, code quality assessment, and testing.

## Skills

| Skill | Trigger | Description |
|-------|---------|-------------|
| **retro** | "run a retro", "reflect on the session" | Session retrospective to capture learnings and improve skills |
| **git-workflow** | "commit", "push", "create a PR", "merge" | Full git workflow orchestration with safety checks |
| **investigate** | "diagnose", "debug", "find the root cause" | Systematic investigation methodology for bugs and mysteries |
| **kaizen** | "find refactoring opportunities", "assess code quality" | Code quality assessment against craftsmanship principles |
| **test** | "test this feature", "verify this works" | Thorough testing with cleanup and go/no-go reporting |

## Installation

Install via the eumemic marketplace in Claude Code:

```
/plugins → eumemic → dev-skills
```
